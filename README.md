# MusicPlayer — 沉浸式音乐播放器

基于 Python + Web 前端的本地音乐播放器，支持双歌词（内嵌/外置LRC）显示、双全屏模式（3D空间/毛玻璃雨景）、实时频谱可视化。

## 启动方式

```bash
pip install flask mutagen
python main.py
```

浏览器打开 `http://127.0.0.1:5000` 即可使用。

## 功能

### 主界面
- 导入歌曲（文件/文件夹/拖拽）
- 播放控制（上下曲、播放模式、进度拖拽、音量）
- 实时频谱可视化
- 右侧歌词滚动显示，当前行高亮居中
- 歌词跟随鼠标滚轮自由浏览，3秒自动回到当前行
- 点击歌词行跳转播放
- 歌词设置面板（字体、颜色、字号、发光、对齐）
- 主题切换（暗黑/霓虹/极简）
- 收藏、播放历史、睡眠定时

### 全屏歌词 — 双模式

点击播放栏 `□` 按钮进入全屏。

**模式1：3D空间**
- 纯白背景，黑色字体+阴影
- 8种出场动画（滑入/淡入/缩放/模糊/3D旋转/打字机/弹跳/飞入）
- 独立设置：字体、颜色、字号(20-96)、阴影深度、动画选择

**模式2：毛玻璃雨景**
- 背景可更换图片
- 毛玻璃 `backdrop-filter` 模糊效果，模糊度可调
- Canvas 雨水粒子特效
- 全部歌词渲染，当前行居中突出
- 独立设置：字体、颜色、字号、模糊强度

**模式切换**：全屏右上角按钮组

## 播放列表

点击顶部 `☰` 打开右侧抽屉，支持：
- 搜索过滤
- 拖拽排序
- 导入/删除歌曲
- 歌词状态标记（内嵌/外置）

## 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| 空格 | 播放/暂停 |
| ↑ ↓ | 音量加减 |
| ← → | 进度后退/快进 5秒 |
| Ctrl+←→ | 上/下一曲 |
| M | 静音切换 |
